## 소개
spring-framework-korean은 제 개인학습 용도로 스프링 소스 코드의 주석(comments)를
한글화 하는 프로젝트 입니다.

2014.12.25 시점으로 프로젝트를 fork하였고 주기적으로 rebase 할 예정입니다.
